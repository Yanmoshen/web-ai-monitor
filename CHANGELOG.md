## [1.0.0] - 2024-11-19

### ✨ 新增 / Added
- 🎉 首次发布 AI Monitor 用户脚本
- 🎯 支持 Gemini 和 ChatGPT 实时响应时间监控
- 📊 智能 Token 数量估算功能(支持中英文混合)
- 💾 本地数据持久化存储
- 🎨 12种精美主题配色方案
  - 蓝色、紫色、粉色、绿色
  - 青色、橙色、红色、黄色
  - 靛蓝、青绿、柠檬绿、灰色
- 🖱️ 可拖拽的悬浮监控面板
- 📈 详细统计报告功能
  - 总体统计(响应次数、总耗时、总Token数、总字符数)
  - 平均数据(平均响应时间、平均Token数、平均字符数)
  - 效率分析(Token/秒、字符/秒)
- 💾 支持导出统计数据
  - JSON 格式导出
  - HTML 报告下载
  - 打印功能
  - 分享功能
- 🔄 一键重置统计数据
- 🌈 二次元风格UI设计
- ✨ 平滑动画效果

### 🛠️ 技术特性 / Technical Features
- 智能检测发送按钮点击
- 支持 Enter 键发送检测
- 防抖处理避免重复监控
- 高效的 DOM 查询优化
- 响应式设计适配不同屏幕
- 实时主题切换无需刷新页面

### 🌐 支持平台 / Supported Platforms
- ✅ Google Gemini (gemini.google.com)
- ✅ ChatGPT (chatgpt.com)
- ✅ OpenAI Chat (chat.openai.com)

### 📝 已知问题 / Known Issues
- 无
